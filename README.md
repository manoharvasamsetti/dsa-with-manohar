# DSA With Manohar - Visual Learning Platform 🚀

An interactive, engaging web application designed to help students and developers visualize how Data Structures and Algorithms work under the hood, step by step. Completely built from scratch without external frameworks using modern HTML, CSS, and Vanilla JavaScript.

🌐 **[Try the Live Web App Here!](replace-with-your-github-pages-link)** *(Add your link here once deployed)*

## ✨ Features
* **Step-by-Step Execution:** Watch algorithms execute line by line with dynamic visual elements.
* **Granular Controls:** Play, Pause, Step Forward, Step Backward, and adjust execution speed dynamically.
* **Pseudocode & Code Tracking:** Live-highlighting of Java code and pseudocode synced precisely with the visual animation.
* **Dark / Light Mode:** Fully responsive, premium UI with toggleable themes.
* **Custom Input:** Generate random data arrays or input your own datasets to visualize.

## 🧠 Algorithms Included

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### Searching Algorithms
- Linear Search
- Binary Search

### Data Structures (Visually Animated)
- Stack (LIFO)
- Queue (FIFO)
- Linked List (Node Insertion)
- Binary Search Tree (Construction & Splitting)

### Graph Algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
